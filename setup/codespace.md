* https://github.com/lichess-org/lila-docker
* Create codespace on main
* Wait for ./lila-docker start to prompt for an input
* Open a terminal on /workspaces/lila-docker (should be one called bash)
* git pull (get latest version of lichess-org/lila-docker
* ./lila-docker setup
* Select advanced (optionally select additional services)
* Wait until you see Lichess homepage in a browser
* You should see folders (repositories) in /workspaces/lila-docker/repos
